# LLM Coding Assistant & Code Reviewer Chatbot

This project is a Streamlit-based chatbot powered by a Large Language Model (LLM).  
The chatbot helps users write code, explain code, debug problems, and review code.

The application provides two modes:
1. Coding Assistant
2. Code Reviewer

---

## LLM Provider

This project uses:

**Google Gemini API**

Model used:
gemini-1.5-flash

---

## Features

- Coding Assistant Mode
- Code Reviewer Mode
- Multi-turn conversation
- Streamlit chat interface
- Sidebar controls (mode selector, API key input, clear chat)
- Error handling
